# Pet Project

uni-app + Vue 3 + Vite H5 project, organized for development on multiple computers with Git.

## Structure

```text
pet-project/
├── frontend/   # uni-app frontend
├── backend/    # reserved
├── docker/     # reserved
├── .gitignore
└── README.md
```

## Frontend

```powershell
cd frontend
npm install
npm run dev:h5
```

The dev server prints the local URL, usually `http://localhost:5173/`.

## Build

```powershell
cd frontend
npm run build:h5
```
